---
title: 1.3 Installing Debian via Live USB
---

Follow these steps for a stable Debian installation:

1. **Flash the ISO** (netinst or live) to USB.
2. **Boot** and select *Graphical Install*.
3. Follow prompts for language, network, user setup.
4. Use guided partitioning (default unless dual-booting).
5. Install the base system and GRUB.
6. Reboot into Debian and log in.
